2. Answer the questions

Given bracket sequence: [((())()(())]]

  • Can this sequence be considered correct?
  • If the answer to the previous question is “no”, then what needs to be changed in it to make it correct?


My answer

First of all, this sequence is not correct

In order to make this sequence correct we need:

  • to add one bracket '[' to the left (if we need a two-dimensional array) or remove one bracket ']' from the right (if we need a one-dimensional array): [[((())()(())]] or [((())()(())]
  • to divide array elements with commas: [[((()),(),(())]] or [((()),(),(())]
  • to remove one bracket '(' from the left side of the first array element or add one bracket ')' to the right side of that element: [(()),(),(())] or [((())),(),(())] or [[(()),(),(())]] or [[((())),(),(())]]
  • All these four examples are correct
